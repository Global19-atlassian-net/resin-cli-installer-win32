Resin CLI Installer Win32
-------------------------

The [Resin CLI](https://github.com/resin-io/resin-cli) Win32 installer.

Prequisites for building
------------------------

- `make`.
- `curl`.
- `makensis`.

Building
--------

Running `make` will create an installer into `build/resin-cli-setup.exe`:

```sh
$ make
```
